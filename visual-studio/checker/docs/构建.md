# 构建说明

> 此项目使必须用 VS 自带的 MSBuild 构建。

## 前置要求
- Visual Studio 2022（Community / Professional / Enterprise）
- 已安装 "Visual Studio extension development" 工作负载
- 将以下目录加入系统 PATH 环境变量：
  - `C:\Program Files\Microsoft Visual Studio\2022\<Edition>\Common7\IDE`
  - `C:\Program Files\Microsoft Visual Studio\2022\<Edition>\MSBuild\Current\Bin`

## 正式构建
> 产物位于 `bin\Release\VsCheckerExtension.vsix`，可直接双击安装或分发。

```powershell
MSBuild VsCheckerExtension.csproj /t:Restore /p:Configuration=Release /v:minimal
MSBuild VsCheckerExtension.csproj /t:Rebuild /p:Configuration=Release /p:DeployExtension=false /v:minimal
```

## 安装
```powershell
# 双击 VSIX 文件即可，或通过命令行静默安装：
VSIXInstaller /quiet bin\Release\VsCheckerExtension.vsix

# 更新配置（必须，否则 VS 不会加载扩展）
devenv /updateconfiguration
```

## 卸载
```powershell
VSIXInstaller /quiet /uninstall:VsCheckerExtension.A59B663D-9E90-46EF-933E-6C2AF9C5FE4E
devenv /updateconfiguration
```

## 调试构建（实验实例）
```powershell
# 构建
MSBuild VsCheckerExtension.csproj /t:Restore /p:Configuration=Debug /v:minimal
MSBuild VsCheckerExtension.csproj /t:Build /p:Configuration=Debug /p:DeployExtension=false /v:minimal

# 安装到实验实例
VSIXInstaller /rootSuffix:Exp /quiet bin\Debug\VsCheckerExtension.vsix
devenv /updateconfiguration /rootsuffix Exp

# 启动实验实例
devenv /rootsuffix Exp
```
